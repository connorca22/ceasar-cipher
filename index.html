Caesar Cipher

ROT Cipher Encryption Tool

Encryption & Decryption Tool

This tool encrypts and decrypts messages using the Caesar Cipher, or "ROT cipher" - which rotates the letters of the alphabet by a specified digit.

Simply select an ROT number to rotate by and encode/decode your message

As an added measure in this particular tool - numbers, punctuation and special characters are also rotated upon encryption/decryption
